# Hook: on_cursor_move

Activated whenever the user moves their cursor within the main haven interface.

## Purpose
Enhance ambience and interactivity through sound and visual responsiveness.

## Behavior
- Pulse the air-flow sound effect based on cursor velocity.
- Emit color-trail particles from cursor anchor point.
- Intensify grass sway slightly near the cursor zone.
- Reduce effects if "Animations Off" mode is active.
- Mute wind sounds if sound toggle is off.

## Notes
This is one of the core interactive elements of the haven's atmosphere.
